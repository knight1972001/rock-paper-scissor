# Rock, Paper, Scissors Game

Welcome to the Rock, Paper, Scissors game! This innovative and interactive application showcases the use of Python, OpenCV, and machine learning to create an engaging gaming experience. Whether you want to play against the machine or challenge your friends, this game has got you covered.

## Features

### 1. Innovative Use of Python and OpenCV

The *Rock, Paper, Scissors* game leverages the power of Python and OpenCV for hand tracking and gameplay. This demonstrates proficiency in computer vision and programming, pushing the boundaries of traditional game development.

### 2. Interactive Gameplay

Choose your mode of play - face off against the machine or challenge your friends for a fun and interactive gaming experience. The game is designed to keep you entertained with its dynamic and responsive gameplay.

### 3. Machine Learning Integration

This game takes it a step further by incorporating machine learning. The application is trained to use a machine learning model, showcasing expertise in developing and integrating AI models. The machine learning aspect is powered by Azure Machine Learning, adding a layer of sophistication to the gaming experience.

### 4. User-Centric Design

We prioritize the user experience, allowing players to choose their opponents and tailor their gaming experience. The user-centric design enhances flexibility and ensures an enjoyable time for everyone.

## Getting Started

1. **Clone the repository.**
   ```bash
   git clone https://github.com/your-username/rock-paper-scissors.git
   ```
2. **Install dependencies.**
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the game.**
   ```bash
   python main.py
   ```
4. **Enjoy the game!** Choose your mode, make your moves, and have fun playing Rock, Paper, Scissors.
